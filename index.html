This is a tool I made, using Python, to solve some of the troubles and inconveniences of keeping and organizing a personal music library. Music folders downloaded from others usually are formatted in a way that is inconsistent to how I format my own library, and this tool allows for looking up artist/album information using the MusicBrainz REST API, then adding ID3 tags to .mp3 files, and formatting track and album naming. This is a simple application, with a GUI made in QT5Designer, for keeping a list of albums, movies, and books that the user might want to listen to, watch, or read. Each list item is added and removed in a first-in-first-out format, making it necessary for the user to be committed to completing whatever is added. GitHub

I made this simple FM synthesizer which uses the Teensy 4.1 microcontroller to check for analog input from the on-board input pins, which is used to modify several parameters (amplitude, frequency, phase) affecting a few waves (which sequentially modulate eachother). PJRC's Teensy Audio Library made this pretty straightforward, but it was a good learning experience for me with the C language and programming MCUs. I also experimented a bit finding different options for audio output (because I initially didn't want to pay $30 or so for the Teensy Audio Shield), and was able to connect it to Ableton Live, where the synthesizer output could be recorded into a project.
